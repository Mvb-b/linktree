# StreamerHub LinkTree Clone

LinkTree personalizado para el streamer StreamerHub.

## 🚀 Descripción

Una página de links estática para el streamer StreamerHub (Misrain Sebastián Valencia Bustos) con diseño dark moderno, estilo gamer/streamer.

## 🎨 Características

- Dark theme con colores negro, púrpura y naranja/coral
- Glassmorphism efectos en los botones
- Animaciones suaves en hover
- Responsive y mobile-first
- Font gaming/streamer (Orbitron)

## 📦 Scripts

```bash
# Instalar dependencias
npm install

# Desarrollo en localhost:3000
npm run dev

# Build estático
npm run build
```

La build estática se genera en la carpeta `dist/`.

## 🔗 Links incluidos

- Twitch
- TikTok
- Instagram
- Twitter/X
- YouTube
- Facebook
